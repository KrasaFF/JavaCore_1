1. Run GameRunner.java
2. Program shows you a field for the game. Each square has a number from 0 to 8.
3. Choose where to put X (enter 0 to 8 from keyboard).
4. After that, computer makes his choice and inputs O.
5. If one of you has 3 in a line, he wins.
6. If the field is filled completely and no lines were found, the game ends in a draw.

Have fun!
